Project Overview
The sinking of the RMS Titanic is one of the most infamous shipwrecks in history. While some passengers survived, many did not. This project uses passenger data to build a machine learning model that predicts survival based on attributes such as age, gender, passenger class, and fare.

Problem Type: Binary Classification
Target Variable: Survived (0 = No, 1 = Yes)
